Project Overview

The Disease Identification Project is a joint effort by the Strong and Agricultural Machinery team, specializing in agricultural engineering at the Faculty of Agriculture, Kafr El-Sheikh University. The aim of the project is to develop a robot that uses deep learning and machine learning to identify diseases on plants in a greenhouse.

The Robot

The robot is designed to move inside a greenhouse and collect images of plants. The robot is controlled using a mobile application that allows the user to steer it in any direction. The robot is equipped with a camera that captures images of plants and sends them to a computer system for analysis.

The Technology

The technology behind the project is based on deep learning and machine learning algorithms that have been trained to identify various diseases that can affect plants. The algorithms analyze the images captured by the robot and classify them based on their characteristics. This allows the robot to identify and differentiate between healthy plants and those that are infected with diseases.

Project Benefits

The Disease Identification Project has several benefits for the agricultural industry. By identifying diseases early, farmers can take action to prevent the spread of the disease and minimize crop losses. This can lead to increased crop yields and higher profits for farmers. Additionally, the project can help reduce the use of pesticides and other harmful chemicals, which can have a positive impact on the environment.
